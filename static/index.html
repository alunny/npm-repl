npm REPL

try node modules, right in your browser

npm REPL allows you to try any npm module[1] without running the laborious and time-consuming process of installing it into your own app[2].

Here are a few samples that work okay, some of the time:

  • Marked — a markdown parser
  • mapleTree — a router
